# 数字

和 JavaScript一样，所有的数字都是浮点数，浮点数的类型是 number ，支持十进制和十六进制以及 ES5中引入的二进制和八进制字面量。

let decLiteral: number = 1;
let hexLiteral: number = 0xf00d;
let binaryLiteral: number = 0b1010;
let octalLiteral: number = 0o744;

# 字符串

使用 string 表示文本数据类型，和 JavaScript一样，可以使用双引号或单引号表示字符串。

let name: string = "zhangsan";
name = "lisi";
1
2

模版字符串 可以定义多行文本和内嵌表达式。这种字符串是被 反引号 包围（ ），并且以${ expr }`这种形式嵌入表达式

let name: string = `zhangsan`;
let age: number = 22;
let sentence: string = `hello, my name is ${ name }.
I'll be ${ age + 1 } years old next month.`;
1
2
3
4
let sentence: string = "hello, my name is" + name + ".\n" + "I'll be" + (age+1) + "years old next month.";

# 数组

有两种方法可以定义数组。

  1. 在元素类型后面接上 [ ]
let list: number[] = [1,2,3];
1
  1. 使用数组泛型 Array<元素类型>
let list: Array<number> = [1,2,3];
1

# 元组 Tuple

元组类型允许表示一个已知元素数量和类型的数组，各元素的类型不必相同。比如，可以定义一对值分别为 string 和 number 类型的数组

let x: [string, number];
x = ['hello',10];
1
2

当访问一个越界的元素，会使用联合类型替代

x[3] = 'world'; // 字符串可以赋值给（string | number）类型
x[6] = true; // ERROR,布尔不是（string | number）类型
1
2

# 枚举

`enum` 类型是对JavaScript标准数据类型的一个补充，可以为一组数值赋予友好的名字。

enum Color{Red,Green,Blue}
let c: Color = Color.Green;
1
2

默认情况从 0 开始为元素编号，也可以手动指定成员的数值，

enum Color {Red = 1,Green,Blue}
let c: Color = Color.Green;
1
2

枚举类型提供一个便利，可以由枚举的值得到它的名字，例如，我们知道数值是2，但不知道它映射到Color的哪个名字，

enum Color {Red=1,Green=2,Blue=3}
let colorName: string = Color[2];
console.log(colorName); // Green
1
2
3

# Any

当不清楚类型的变量，同时不希望类型检查器对这些值进行检查而是直接让它们通过编译阶段的检查。可以使用 any 类型来标记

let notSure: any = 4;
notSure = 'maybe a string';
notSure = false;
1
2
3

object 有类似的作用，但只是允许给它赋任意值，但却不能在它上面调用任意的方法，即便真的有这些方法

let notSure = 4;
notSure.toFixed(); // ok
let prettySure: Object = 4;
prettySure.toFixed(); //Error
1
2
3
4

当只知道一部分数据的类型时，比如有一个数组包含了不同类型的数据

let list: any[] = [1,true,'zhangsan'];
list[1] = 100;
1
2

# Void

void 和 any 类型相反，表示没有任何类型，当一个函数没有返回值时，

function something(): void{
    console.log('this is a function');
}
1
2
3

声明一个 void 类型的变量没有什么大用，因为只能为它赋予 undefined 和 null

let unusable: void = undefined;
1

# Object

object表示非原始类型，也就是除 number , string , boolean ， symbol ， null 或 undefined 之外的类型。

declare function create(o: object | null):void;
create({ prop: 0 }); // OK
create(null); // OK
create(42); // Error
create("string"); // Error
create(false); // Error
create(undefined); // Error
