# src和href有什么区别

  • src用于替换当前元素；href用于在当前文档和引用资源之间建立联系；

  • src是source的缩写，指向外部资源的位置，指向的内容将会嵌入到文档中当前标签所在位置； 而 href 是 Hypertext Reference 的缩写，指向网络资源所在的位置，建立和当前元素或当前文档之间的链接；
